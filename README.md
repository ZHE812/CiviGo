# 🏛️ CiviGo — 每天走一點，文明前進一點

[![Demo Status](https://img.shields.io/badge/Status-MVP%20Demo-gold)](https://zhe812.github.io/CiviGo/)
[![Competition](https://img.shields.io/badge/Event-2026%20桃園青創力競賽-blue)](https://zhe812.github.io/CiviGo/)

> **「你的每一步，都是人類文明進步的一大步。」**
> CiviGo 是一款結合「遊戲化文明進化」與「AI 飲食分析」的健康管理平台。我們將枯燥的步數數據轉化為建設文明的資源，讓健康生活變成一場史詩級的冒險。

---

## 🌟 核心理念

傳統的健康 App 往往因缺乏長期驅動力而導致用戶留存率低下。CiviGo 透過 **「文明演進」** 的機制，將使用者的健走步數與飲食狀況轉化為「勞動力」、「糧食」與「科技值」，藉此解鎖從「史前時代」到「工業時代」的各種建築與里程碑。

---

## ✨ 核心功能

### 1. 📊 文明儀表板 (Civilization Dashboard)
* **步數即資源**：將每日步數即時轉換為文明等級（Civ Score）與勞動力。
* **文明爆發機制**：當達成每日步數目標且完成 AI 飲食分析時，將觸發「文明爆發」，獲得額外的科技值加成。
* **連續達標獎勵**：追蹤連續達標天數，強化用戶的心理留存。

### 2. 🤖 AI 飲食分析 (Powered by Gemini Vision)
* **即時影像辨識**：上傳食物照片，透過 Gemini AI 分析熱量、蛋白質、碳水與脂肪。
* **文明影響評估**：AI 會根據飲食的均衡程度，給予「糧食/穩定度」獎勵或建議步數，將飲食選擇與遊戲數值掛鉤。

### 3. 🏛️ 時代演進與建築解鎖
* **四大時代**：史前時代 🪨 → 農業文明 🌾 → 青銅城邦 ⚔️ → 工業時代 ⚙️。
* **動態建築系統**：隨著文明值提升，自動解鎖如「火堆」、「農田」、「神殿」到「發電站」等 12 種以上特色建築。

### 4. 🏆 聯賽與全球排行
* **青銅文明聯賽**：每週結算的聯賽機制，設有晉升與降級區，激發社群競爭心。
* **全球名人堂**：展示頂尖「文明行者」的時代等級與總積分。

## 💡 創業構想 (Why CiviGo)

* **創新性**：首創將健康行為與 4X 策略遊戲（探索、擴張、開發、征服）機制深度結合。
* **市場可行性**：鎖定大學生與 25–35 歲年輕上班族，利用社群分享與同儕競爭降低健康行為的門檻。
* **商業模式**：
    * **Freemium**：基礎功能免費。
    * **訂閱制**：進階 AI 飲食分析報告。
    * **B2B 授權**：提供校園健康挑戰賽、企業員工健康管理方案。
    * **品牌聯名**：與品牌合作推出限定版文明建築。

## 🛠️ 技術架構

* **Frontend**: HTML5, CSS3 (Variable, Grid, Flexbox, Animations), JavaScript (Vanilla ES6+).

* **UI/UX**: 採用 Dark Mode 深色質感設計，搭配高飽和度霓虹點綴與毛玻璃特效 (Backdrop Filter)。

## 🚀 快速上手

### 如何執行 Demo？
1.  下載 `index.html` 檔案。
2.  直接使用瀏覽器開啟該檔案。
3.  點擊右上角的 **「🎬 Demo 展示模式」**，系統將引導你體驗完整流程（包含飲食分析、步數增加、文明爆發到排名提升）。

### 如何串接 AI 功能？
1.  前往 [Google AI Studio](https://aistudio.google.com/) 申請免費的 Gemini API Key。
2.  在頁面中的「AI 飲食分析」區塊貼上你的 API Key。
3.  上傳一張食物照片並點擊「執行分析」。
